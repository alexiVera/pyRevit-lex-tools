# pyRevit LEXBIM Tools

Colección de automatizaciones para Revit usando pyRevit.

## Extensiones incluidas

- Extraer-datos → extrae datos de elementos del modelo en revit en una matriz xml
- Actualizar-datos → Actualiza los datos de la matriz modificada en xml

## Instalación

pyrevit extensions add https://github.com/alexiVera/pyRevit-lex-tools.git
